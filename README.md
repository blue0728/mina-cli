# mina-cli
小程序构建工具

# install
```
npm install mina-cli2 -g  全局安装 mina-cli2
```

```
mina -v   版本号
mina create [项目名称]  创建项目
```
![install](http://p7inyv4vd.bkt.clouddn.com/1)

```
cd test && npm install   安装项目
npm run dev    运行项目
npm run create   新建页面
```
